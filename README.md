# 83 Rifle Range Rd — Guest Guide

A private digital guest guide for **83 Rifle Range Road, Narangba, Queensland**.

Deployed via GitHub Pages. Shared with guests via QR code.

## Pages

| File | Purpose |
|------|---------|
| `index.html` | Main guest guide — location, essential info, contacts |
| `orientation-checklist.html` | Arrival walkthrough checklist |
| `exit-checklist.html` | Departure checklist |

## Structure

```
/
├── index.html
├── orientation-checklist.html
├── exit-checklist.html
├── style.css
├── robots.txt
└── resources/
    ├── Hero.jpg
    ├── *.jpg              (gallery photos)
    └── Home Exchange Manual New.pdf
```

## Deployment

Hosted on GitHub Pages from the `main` branch root.

URL: `https://pakton-paul.github.io/RR-Rd-Website/`

## Privacy

- All pages carry `<meta name="robots" content="noindex, nofollow">`
- `robots.txt` blocks all crawlers
- Do not include security codes, passwords, or sensitive data in commits

## Access

Guests access the site by scanning a QR code provided on arrival.
